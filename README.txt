VidaPlena SpA - Landing Page
Descripción del Proyecto
Este proyecto consiste en el desarrollo de una Landing Page (página de aterrizaje) estática para VidaPlena SpA, un centro orientado al bienestar integral holístico. El diseño busca transmitir calma y profesionalismo a través de una paleta de colores basada en tonos verdes y grises claros, ofreciendo a los usuarios información clara sobre los servicios de yoga, meditación y nutrición consciente.

El proyecto fue construido con un enfoque educativo, aplicando buenas prácticas de desarrollo web, código limpio y estructurado.

Tecnologías Utilizadas
HTML5: Uso de etiquetas semánticas (<header>, <section>, <nav>, <footer>) para una mejor estructura y accesibilidad.

CSS3:

Implementación de Flexbox para la alineación y distribución responsiva de los elementos.

Uso de Variables CSS (:root) para mantener la consistencia en la paleta de colores y facilitar futuras modificaciones.

Efectos de transición suaves (hover) para mejorar la interactividad visual de los botones y tarjetas.

Estructura de Secciones
La página es de tipo One-Page (una sola página) y cuenta con navegación interna mediante enlaces de ancla (anchor links). Sus secciones principales son:

Header (Navegación): Barra superior fija (sticky) que incluye el logotipo de la marca y enlaces a las distintas secciones de la página.

Hero Section: Sección principal de impacto con el mensaje central de la marca, una imagen representativa y un llamado a la acción (Call to Action).

Servicios y Beneficios: Área de contenido visual (imagen general) seguida de tarjetas (cards) descriptivas para los servicios de Yoga, Meditación y Nutrición.

Testimonios: Sección que destaca las opiniones de clientes para generar confianza (prueba social).

Contacto: Formulario de captura de datos (maqueta visual) diseñado para que los usuarios interesados puedan comunicarse.

Footer: Pie de página con información de contacto local, enlaces a redes sociales y derechos de autor.

USO de IA: Gemini para corrección de ajustes visuales al cargar imágenes.
Prompt: Ayudame a ajustar visualmente la imagen logo_vidaplena_spa.png en el área de navegación como logo visible para el header.
Prompt: Corrige error visual del area de bienestar al momento de cargar imagen, desde la dirección de archivo img/
